# Bachelor in Data Science - Projects Portfolio
## SUPSI Lugano (Switzerland)

This repository serves as a comprehensive collection of academic projects and research conducted during my Bachelor's degree in **Data Science** at **SUPSI** (Scuola Universitaria Professionale della Svizzera Italiana) in Lugano.

The projects are organized by academic year and reflect a progression from fundamental data analysis to advanced machine learning, robotics, and complex software systems.

---

## 📂 Project Catalog by Theme

### 📊 Data Science & Exploratory Analytics
*   **World Energy Analysis (Year 1)**: Extensive analysis of global energy consumption patterns, utilizing data visualization and statistical tools to identify trends in energy usage and sustainability.
*   **Data Science Hackathons (Year 1 & 2)**: Intensive data challenges where I applied data cleaning, feature engineering, and predictive modeling under tight deadlines.

### 🤖 Artificial Intelligence & Machine Learning
*   **Machine Learning (Supervised & Unsupervised)**: Implementation of various models including linear regression, decision trees, clustering (K-Means, PCA), and ensemble methods.
*   **Advanced Machine Learning (Year 3)**: Deep learning applications and advanced optimization techniques for complex datasets.
*   **Bayesian Data Analysis (Year 3)**: Probabilistic modeling and inference using Bayesian statistics to handle uncertainty in data.
*   **Computer Vision (Year 3)**: Image processing and object detection tasks using Convolutional Neural Networks (CNNs).
*   **Fundamentals of Bioinformatics (Year 3)**: Application of data science techniques to biological data, including sequence alignment and genomic analysis.

### 💻 Software Engineering & Data Systems
*   **Data Management Project (Year 2)**: Design and implementation of database systems, focusing on SQL/NoSQL architectures and data integrity.
*   **Web Service Task Management (Year 2)**: Development of a full-stack or API-based web service for task orchestration and management.
*   **Robot Control (Year 3)**: Programming and control of robotic systems, focusing on path planning and sensor integration.

### 📈 Specialized Data Analysis
*   **Time Series Analysis (Year 3)**: Forecasting and analysis of temporal data, applying models like ARIMA and LSTM for trend prediction.
*   **Trustworthy Autonomous Systems (Year 2)**: Research and development focused on the reliability, safety, and ethics of autonomous agents.
